# Bank Management System

## Description
A console-based Bank Management System developed using Python and Object-Oriented Programming (OOP).

## Features
- Create Bank Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Check Balance
- Change PIN
- View Account Details
- Transaction History
- Change Account Holder Name
- Input Validation
- Encapsulation using private attributes

## Technologies Used
- Python 3

## Concepts Used
- Classes & Objects
- __init__()
- Encapsulation
- Instance Variables
- Instance Methods
- Loops
- Conditional Statements
- Lists
- Functions
- Modular Programming

## Project Structure

Bank-Management-System/
├── bank.py

├── main.py

## How to Run

```bash
python main.py
```
